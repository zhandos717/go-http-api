## API

#### /payment
* `POST` : Create payment

#### /payments
* `GET` : Get all payments


#### /payment/:id
* `GET` : Get payment by id
* `PUT` : Edit payment by id
* `DELETE` : Delete payment by id


#### /category
* `GET` : Create category

#### /categories
* `GET` : Get all categories

#### /category/:id
* `GET` : Get category by id
* `PUT` : Edit category by id
* `DELETE` : Delete category by id
